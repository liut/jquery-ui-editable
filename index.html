jQuery UI - Inline Edit

Inline Edit is a jQuery UI plugin, which will hopefully be part of the official package, designed to make inline editing as seamless and as easy as possible.

  1. Triggers on Double Click, finished on input blur or enter press.
  2. Triggers on Single Click, finished on enter press only.
  3. Triggers on Double Click, must have a number to pass validation. (1)
